# Customer Segmentation using K-Means

This project performs customer segmentation using K-Means clustering.
It groups customers based on annual income and spending score.
The problem is unsupervised since no labels are provided.
Feature scaling is applied to ensure balanced clustering.
The Elbow Method is used to determine the optimal number of clusters.
The model identifies distinct customer groups based on behavior.
Clusters help understand spending patterns across customers.
Results are visualized using scatter plots.
This can be used for targeted marketing and business insights.
The project demonstrates basic unsupervised learning concepts.
